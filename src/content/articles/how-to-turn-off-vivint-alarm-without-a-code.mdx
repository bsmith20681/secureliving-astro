Are you wondering how to disarm a Vivint alarm without a code? Did you just move into a [house with a Vivint alarm system](https://secureliving.com/home-security-systems/bought-a-house-with-vivint/)? Are you trying to figure out how to get it to stop beeping at you? Or maybe you want to start using the system to protect your home?

I put this simple guide together to walk you through how to turn Vivint off without an alarm code.

## Use a Common Pin Code or Get the Pin Code from the Previous Owner

In order to turn the alarm off, you’ll obviously need to have the pin code. If you were to call Vivint and ask for a pin code they unfortunately not give it to you unless you are a customer (more on this later).

In this situation, you can either reach out to the previous owner for the pin code or use guess what the pin code. Sounds crazy, but you would be surprised how often people use simple pin codes.

Here are **4 common pin codes** you can try to turn the alarm off.

- **1111**
- **2680**
- **0000**
- **2203** (This used to be a common installer code for techs)

If neither of those pin codes work and you can’t get ahold of the previous owner. Then your options are to either remove the battery or to call Vivint to active the account.

## Call Vivint to Create an Account

Unfortunately, if you call Vivint they are not going to tell you the pin code to turn off the system **unless** you are a paying customer.

The good news is that since you have the system already installed creating your account and activating the system can all be done over the phone. They do require that you pay for the first month which starts at $29.99 per month for just standard service. The service is month-to-month. So there are no contracts.

If you subscribe to Vivint’s monitoring service, they will create an account for you. Assuming you live on the property where the alarm system is blaring, this will allow you to instantly turn off the alarm. In fact, from now on, you’ll be able to turn off the alarm whenever you want.

## Disarm Vivint without a code by Removing the Battery from the Panel 

<Image src={vivintBackPanel} alt="back of vivint panel" />

Remove the panel by pushing up and over. Then pull the tab out to remove the battery.

If you’re really struggling and you’ve got an alarm blaring in the middle of the night, then you’ve got one more solution. That’s to learn how to turn off Vivint panel. Obviously, you can’t do this with the pin code, so you need to cut the power.

This means removing the battery from the panel. It doesn’t take too long. It’ll turn the alarm off, though. Just bear in mind that you’ll need to come up with a better solution before you replace that battery again. This is just a way to silence the alarm in the immediate moment.

<Admonition>
  If the alarm is armed and you go to remove the battery the alarm will be
  triggered! When the battery is removed then it will turn off.
</Admonition>

Hopefully, one of these solutions worked. Remember, the only way to get a long-term fix for a forgotten pin code is to contact Vivint and subscribe to their service.
