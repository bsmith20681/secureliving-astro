Did you just move into a house with ADT? You're not alone. This is one of the most common questions I get from people buying a home with an existing ADT security system already installed. In this article, I cover what to do with your existing ADT equipment, how to turn the ADT alarm on and off, get a pin code, and how to activate the ADT monitoring service if that is what you are interested in.

Since the ADT security equipment is already in your home, this can help save you money since you can use what's already installed. In this guide, we'll go over how to activate the ADT system, use the ADT equipment without professional monitoring, and explore upgrade options if you're looking for smart home security features.

## How do I get the ADT system turned on and get a pin code?

In order to get the ADT security system turned on, you will need to call in to activate the service. Their phone number is <a href={`tel:${companyData.phoneUnformatted}`}>{companyData.phone}</a> and you can also check out their <a href={companyData.website}>website</a>. When you call ADT, you will speak to a sales rep over the phone who can set up your account, go over ADT packages, and discuss pricing for your home security monitoring.

ADT's monthly monitoring cost starts at around $42.99 per month for their base package. If you want ADT cameras and smart home security features, the pricing goes up to about $48.99 per month.

Once you have decided on an ADT monitoring package, they will schedule a time and date to have an installer come to your home to activate the security system. Since you already have the ADT equipment installed, you can avoid the setup fee. Once the ADT installer reactivates the home security system, you will be given your pin code and taught how to use it.

From there, you will be able to successfully turn the ADT alarm on and off.

<Admonition>
  ADT requires a monitoring contract even if you already own the equipment. The
  minimum ADT contract is 12 months. If you finance the equipment, the contract
  length matches the financing period, up to 36 months. There is no way around
  this.
</Admonition>

The only benefit to having the existing ADT equipment inside of the home is that you don't have to pay for a setup or installation fee. The monitoring contract and monthly cost will be the same as any other ADT customer.

## Can I use ADT equipment without monitoring?

Yes, you can use ADT equipment without professional monitoring. The ADT security system will still sound the siren even if you are not paying for the monitoring service.

If you are able to get the pin code from the previous owners, then you can use the ADT system like normal. This means that you can arm and disarm the ADT security system with that pin code. The sensors, siren, and life safety devices like smoke detectors will still function.

If you have the ADT system armed and someone were to try and break in, the siren will still sound like normal. Since the ADT alarm is going off, no burglar is going to stick around to see if the cops show up!

The features you lose out on if you don't pay for ADT professional monitoring are the ability to use the ADT app on your phone and for ADT to dispatch police, fire, or medical services if the alarm is triggered. The siren, sensors, and life safety equipment will still work without monitoring.

If you want to learn more about the ADT service, you can check out my [ADT review here](/home-security-systems/adt-review/).

## What if I don't have the pin and I don't want monitoring?

Your best bet on getting the pin code, if you don't already have it, is to reach out to the previous home owner. This is one of the most common questions people have when buying a home with an existing ADT security system.

If you call ADT, they will not give any pin code out over the phone even if the ADT system is no longer active. This is a safety measure that ADT has in place to protect previous customers.

If you can't get a hold of the previous owner, here are some common pin codes where you might have some luck trying.

You can try..

- 1234
- 2468
- 7923

None of these codes are guaranteed to work. However, they are common pin codes that you can try.

If the ADT security system keeps beeping at you, you will need to uninstall the panel or just start up ADT <a href={companyData.website}>monitoring service</a>.

## ADT packages and pricing

One of the first questions people ask is how much ADT monitoring costs when they already have the equipment in their home. Here's a breakdown of ADT's current packages and pricing.

ADT offers two main monitoring packages for home security:

- **ADT monitoring at $42.99/month** includes 24/7 professional monitoring, ADT security app access, smart home automation, and life safety monitoring for smoke, carbon monoxide, and flood sensors.
- **ADT monitoring with cameras at $48.99/month** includes everything above plus ADT camera services with live view, cloud storage, and smart security alerts.

Since you already have ADT equipment in your home, you can save on the upfront equipment cost. The setup and installation are the main things you can avoid paying for. However, the monthly monitoring pricing will be the same.

ADT does require a monitoring contract with a minimum of 12 months. If you finance any new ADT equipment, your contract extends to match the financing terms. This is one thing I wish ADT would change, but every major security company with professional installation operates this way.

If you want a full breakdown of what each ADT plan includes and which one makes sense for your situation, read my [guide to ADT monitoring plans for new homeowners](/home-security-systems/adt-monitoring-plans-for-new-homeowners).

## Smart home and life safety features with ADT

If the ADT system in your new home is older, you might be missing out on some of the smart home and safety features that ADT now offers. Here's what you can get with a modern ADT security setup.

**Smart home security features** include controlling locks, lights, thermostats, and garage doors from the ADT app. You can set up smart automations so that when you arm the ADT security system, all the doors lock and the lights turn off. ADT uses Z-Wave smart home devices, and the system is also compatible with Nest products.

**Life safety monitoring** is something a lot of people overlook. ADT can monitor smoke detectors, carbon monoxide detectors, and flood sensors in your home. If a life safety device is triggered, ADT's monitoring center dispatches fire or medical services to your home. This is especially important for families with kids.

**ADT cameras** are another upgrade worth considering. ADT offers indoor, outdoor, and doorbell cameras. All ADT cameras include HD video, night vision, two-way talk, and 30 days of cloud storage. If you're buying new ADT security equipment, I'd recommend at least getting a doorbell camera and one outdoor camera for your home.

ADT has been the largest home security company for a long time, so chances are that you have older ADT equipment in your home.

Most people are ok with a more traditional home security system without all the bells and whistles. The only thing that I would recommend is making sure that the ADT system is cellular based. This means that ADT monitoring will call out to the police department using cell towers instead of your landline.

## Can I use this ADT equipment with another security company?

Most ADT equipment can be used with another security company. Your ADT panel and any ADT security cameras will not work with other companies. Those will be replaced by the new security company. However, any wired sensor that is in the home will work with the following companies.

- Vivint
- SimpliSafe (requires their own sensors, but will install alongside existing wired systems)
- Ring
- ADP Security
- Any other local security company that uses 2Gig, GE, or Honeywell equipment

Some wireless ADT sensors will not work with these security systems. It is always a good idea to call in advance to ask about compatibility before signing a contract.

If you're thinking about switching from ADT to another home security company, I'd recommend looking at Vivint or SimpliSafe as alternatives. Vivint offers similar professional monitoring and smart home services with comparable packages. SimpliSafe is a good option if you want a home security system without a long-term contract, since they offer month-to-month monitoring.

## I accidentally triggered the ADT alarm, how do I turn it off?

If you set the ADT alarm off by accident then you will want to put in the pin code to turn it off.

If you don't have the pin code then you will want to unplug the ADT panel and remove the battery.

If you just unplug the ADT panel, the backup battery will still power the security system for another 3-4 hours at least. Here are some instructions on how to [replace the battery](https://www.adt.com/help/faq/security-services-features/replace-system-battery). These steps will help you to actually remove the battery from the ADT security system.
